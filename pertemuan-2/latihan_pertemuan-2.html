Pengenalan HTML


Apa itu HTML?


HTML adalah singkatan dari HyperText Markup Language.Ini adalah bahasa dasar yang digunakan untuk membuat halaman web.Semua website yang kamu lihat di internet-dari situs berita sampai media sosial-semuanya dibuat menggunakan HTML sebagai pondasinya.

HTML tidak dianggap sebagai bahasa pemrogaman karena dia tidak bisa melakukan perhitungan logika seperti menghitung angka atau menjalankan fungsi.Tapi tanpa HTML,halaman web tidak akan punya struktur sama sekali.


Fungsi HTML


HTML berfungsi untuk memberi struktur dan makna pada konten di halaman web.Misalnya,tag h1 digunakan untuk membuat judul utama,tag p untuk paragraf,dan tag img untuk menampilkan gambar.
Dengan HTML,browser tahu mana yang harus ditampilkan sebagai teks besar,mana yang jadi paragraf,dan mana yang berupa tautan atau gambar.


Kenapa harus belajar HTML?


Belajar HTML adalah langkah pertama buat siapapun yang ingin jadi web developer.Dengan memahami HTML,kamu akan lebih mudah belajar CSS untuk mempercantik tampilan dan JavaScript untuk menambah interaktivitas.
Bahkan kalau kamu tidak ingin jadi progammer,mengerti HTML tetap berguna-misalnya untuk menulis artikel di blog,membuat email newsletter,atau sekedar tahu bagaimana internet bekerja.


Kesimpulan


HTML adalah dasar dari dunia web.Tanpa HTML,tidak akan ada halaman web seperti yang kita kenal sekarang.Jadi,kuasai dulu sturktur dan tag tag dasarnya.Setelah itu,baru lanjut ke CSS dan JavaScript untuk membuat website yang lebih menarik.
