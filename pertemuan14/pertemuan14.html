form elemen html

pertemuan kali ini akan menjelaskan perbedaan elemen yang digunakan dalam form html

html form elemens dapat mencakup lebih dari satu elemen yang dapat digunakan

  • input

    • elemen yang sering digunakan dalam form. Elemen input dapat ditampilkan dalam berbagai cara, tergantung dari atribut type nya. COntoh

  • label

    • elemen label mendefinisikan beberapa elemen form. Elemen label bberguna bagi pengguna screen reader karena screen reader akan memebacakan label ketika pengguna memfokuskan elemen input. elemen label juga membantu pengguna yang kesulitan meng klik area yang sangat kecil(seperti tombol radio atau checkbox) karena ketika pengguna mengklik teks didalam label akan mengaktifkan tombol radio atau kotak centang. atribut for pada tag label harus sama dengan atribut id pada elemen input untuk menghubungkan kedua nya.

  • select

    • elemen select mendefinisikan drop down list, artinya elemen option akan memberikan beberapa pilihan yang dapat dipilih. Secara default item pertama dalam drop down list akan terpilih. Contoh

  • text area

    • elemen text area mendefinisikan area teks yang dapat digunakan untuk memasukkan teks dalam jumlah yang lebih banyak dibandingkan dengan elemen input type="text". Contoh

  • button

    • elemen button mendefinisikan tombol yang dapat di klik

  • legend
  • data list
