HTML input atribut

HTML input atribut adalah atribut yang digunakan untuk memberikan informasi tambahan pada elemen input dalam HTML. Atribut ini dapat digunakan untuk mengatur perilaku, tampilan, dan validasi dari elemen input. Beberapa contoh atribut yang umum digunakan pada elemen input adalah:

  1. value atribut

    value atribut pada elemen input menentukan nilai awal utuk field input



  2. atribut readonly

    Atribut readonly pada elemen input menentukan bahwa field input bersifat hanya dibaca field input yang hanya bisa dibaca tidak diubah(namun pengguna masih bisa menavigasi, menyorot dan menyalin teks darinya). Nilai dalam field imput yang bisa dibaca akan dikirim saat for nya di submit.



  3. Disabled atribut

    atribut disabled pada elemen input menentukan bahwa field input harus di nonaktifkan.

    field input yang dinonaktifkan tidak dapat digunakan dan tidak dapat di klik.Nilai dari field input yang dinonaktifkan tidak akan dikirim saat form di submit.



  4. atribut required

    atribut required pada elemen input menentukan bahwa field input harus diisi sebelum form dapat disubmit.

    atribut required berfungsi pada tipe input berikut: text,search,url,tel,email,password,date pickers,number,checkbox,radio,file
