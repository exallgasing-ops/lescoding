form atribut

kali ini kita akan mendefinisikan atribut dalam html pada elemen form

  1. action atribut

    2. atribut action menentukan tindakan yang akan dilakukan saat form dikirimkan. Biasanya, data form akan dikirimkan ke sebuah berkas di server ketika pengguna mengklik tombol submit.

  2. target atribut

    3. atribut target menentukan dimana respons yang diterima setelah menguirimkan form akan ditampilkan. atribut target dapat memeliki salah satu nilai berikut:

    value deskripsien
    _blank respons akan ditampilkan di jendela baru atau tab baru
    _self respon ditampilkan di jendela saat ini
    _parent respons ditampilkan di frame induk
    _top respons ditampilkan di seluruh badan jendela
    framename respons ditampilkan di iframe yang bernama

    default value dari atribut target adalah _self yang artinya semua respon akan ditampilkan di jendela saat ini

  3. method atribut

    4. atribut method menentukan metode HTTP yang digunakan saat mengirim data form. data form dapat dikirimkan sebagai variabel URL (dengan method="get") atau sebagai transaksi http POST (dengan method="post") metode http default saat mengirimkan data form adalah get
